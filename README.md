## How to Run Scripts

Make scripts executable:

chmod +x scripts/*.sh

Run scripts:

./scripts/system_identity.sh
./scripts/package_inspector.sh
./scripts/disk_audit.sh
./scripts/log_analyzer.sh /var/log/syslog
./scripts/manifesto.sh
